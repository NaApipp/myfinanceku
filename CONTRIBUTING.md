# Panduan Kontribusi — MyFinanceKu

Terima kasih telah tertarik untuk berkontribusi pada **MyFinanceKu**! Kami sangat menghargai bantuan Anda untuk membuat aplikasi ini menjadi lebih baik.

## 🚀 Cara Mulai Kontribusi

1.  **Cari Issue**: Lihat daftar issue kami atau buat issue baru jika Anda menemukan bug atau memiliki ide fitur.
2.  **Fork & Clone**: Fork repositori ini dan clone ke mesin lokal Anda.
3.  **Buat Branch**: Gunakan nama branch yang deskriptif, contoh: `feat/fitur-baru` atau `fix/bug-nama`.
4.  **Implementasi**: Lakukan perubahan kode Anda. Pastikan untuk mengikuti standar coding yang ada.
5.  **Commit**: Gunakan [Conventional Commits](https://www.conventionalcommits.org/) (contoh: `feat: tambah grafik pengeluaran`).
6.  **Pull Request**: Kirim PR Anda dengan penjelasan mendetail tentang perubahan yang dilakukan.

## 🛠 Standar Coding

-   **TypeScript**: Semua file baru harus menggunakan TypeScript. Hindari penggunaan `any`.
-   **Komponen**: Gunakan Functional Components dengan Tailwind CSS untuk styling.
-   **Linting**: Pastikan kode Anda lulus cek linting dengan menjalankan `npm run lint`.

## 📝 Pesan Commit

Kami mengikuti konvensi berikut:
-   `feat:` untuk fitur baru.
-   `fix:` untuk perbaikan bug.
-   `docs:` untuk perubahan dokumentasi.
-   `style:` untuk perubahan formatting (whitespace, missing semi-colons, dll).
-   `refactor:` untuk perubahan kode yang tidak memperbaiki bug maupun menambah fitur.
